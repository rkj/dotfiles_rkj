Various scripts and configuration files I use.

To get started:
```sh
$ git clone git@github.com:rkj/dotfiles_rkj.git
$ cd dotfiles_rkj
$ ./bin/provision
$ ./init.sh
```

To add new files into the repo use:
```sh
$ ./adopt.sh ~/.file_name
$ ./adopt.sh --force ~/.file_name # overwrite an existing repo destination
```
